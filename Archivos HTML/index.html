Toma de Ramos

Entrega Docs

En este repositorio se encuentran los archivos html y css del proyecto "Toma de Ramos" de la Universidad del Desarrollo.
Esta es una entrega previa para que pueda ser revisada antes de la entrega final, por ende está sujeta a cambios y actualizaciones.
Las vistas están adaptadas a móviles, se debe abrir la ruta en el formato móvil para ser revisada.
Este repositorio será eliminado una vez que sea entregado el proyecto.
